# TechShop Pro - Simulação de E-commerce 

Este projeto é uma aplicação web completa de frontend que simula uma loja online de produtos tecnológicos. [cite_start]O objetivo principal foi demonstrar o domínio das tecnologias core do desenvolvimento web: **HTML5, CSS3 e JavaScript**.

## Funcionalidades Principais

* [cite_start]**Catálogo Dinâmico**: Visualização de produtos tecnológicos com sistema de filtragem.
* [cite_start]**Carrinho de Compras**: Sistema funcional para adicionar produtos, atualizar quantidades e remover itens, com cálculo do total em tempo real.
* [cite_start]**Sistema de Favoritos**: Permite marcar produtos de interesse com atualização dinâmica do contador na navegação.
* [cite_start]**Persistência de Dados**: Utilização de `localStorage` para garantir que os dados são mantidos ao navegar entre páginas.
* [cite_start]**Design Responsivo**: Layout adaptável para Desktop, Tablet e Mobile através de CSS3.
* [cite_start]**Validação de Formulários**: Verificação de campos obrigatórios no formulário de registo.

## Tecnologias Utilizadas

* [cite_start]**HTML5**: Estrutura semântica utilizando etiquetas como `<header>`, `<nav>`, `<main>`, `<section>` e `<footer>`.
* [cite_start]**CSS3**: Estilização moderna com efeitos de hover, transições e sombras.
* [cite_start]**JavaScript**: Lógica de negócio, manipulação do DOM e gestão do estado via `localStorage`.

## Estrutura do Projeto

[cite_start]A aplicação está organizada em 7 páginas distintas para garantir uma navegação fluida:
1. [cite_start]**Home**: Ponto de entrada focado na conversão.
2. [cite_start]**Produtos**: Catálogo completo.
3. [cite_start]**Promoções**: Secção de ofertas e descontos.
4. [cite_start]**Favoritos**: Gestão de itens desejados.
5. [cite_start]**Carrinho**: Gestão de compras e finalização de pedido.
6. [cite_start]**Registo**: Formulário de criação de conta.
7. [cite_start]**Contacte-nos**: Canal de suporte e contacto.
